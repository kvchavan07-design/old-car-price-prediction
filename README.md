# 🚗 AI Used Car Price Prediction

A Machine Learning web application that predicts the selling price of used cars based on vehicle specifications. The application is built using **Python**, **Scikit-learn**, and **Streamlit**, and uses a **Random Forest Regressor** trained on historical used car data.

---

## 📌 Features

- 🚗 Predict used car selling price instantly
- 🤖 Random Forest Regressor model
- 📊 Interactive Streamlit web interface
- 📋 Vehicle summary after prediction
- 📈 Model performance dashboard
- 💡 Price category analysis
- 🎨 Responsive and modern UI

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

---

## 📂 Dataset

- **Number of Records:** 7,907
- **Number of Features:** 46
- **Target Variable:** Selling Price

### Input Features

- Brand
- Fuel Type
- Seller Type
- Transmission
- Owner
- KM Driven
- Mileage
- Engine Capacity
- Maximum Power
- Number of Seats
- Car Age

---

## 🤖 Machine Learning Model

**Algorithm**

- Random Forest Regressor

### Model Performance

| Metric | Value |
|---------|--------|
| Test R² Score | **96.39%** |
| Cross Validation R² | **94.50%** |
| RMSE | **₹164,264** |

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Used-Car-Price-Prediction.git
```

Go to the project folder

```bash
cd AI-Used-Car-Price-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
